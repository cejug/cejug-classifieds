What is it? CEJUG-Classifieds is a 100% pure Java classified advertising systems.

Project Goals

The goal of the project is to create the infra-structure to support a classified advertising system, including three sub-systems:

  1. Middleware: a web-service based system responsible by:
    • The business logic.
    • Users management.
    • Interoperability with third party systems.
  2. Persistence: an ORM system used to persist the data - it will be based on a popular framework, not yet selected.
  3. GUI: a end customer application planned to be the most human-friendly and efficient way to access the classified advertising system.

Downloads

You can try the latest version of the web-service prototype. Instructions:

  1. Install Glassfish
  2. Download the cejug-classifieds-server.war
  3. Start default server
  4. Copy the war file to the directory <%GLASSFISH_HOME%>\domains\domain1\autodeploy
  5. Open a browser with the URL: http://localhost:8080/cejug-classifieds-server/server

* If the service does not load, you can check the LOG file at <%GLASSFISH_HOME%>\domains\domain1\logs\server.log

Every day we are exposed to a bunch of brand new technologies and sometimes we simply don´t have time to read about its design and applicability. Several frameworks and tools quickly marked trends and all this commercial rhythm causes a lack of understanding about the underneath techniques of each new stuff. We have listen about Struts, Spring, Hibernate and several other products without much time to discuss and learn about how they really work. Frameworks are powerful tools and can help us to produce a complete project in few weeks instead of several months. Complex IDEs also give us the idea of programming as a press the button task.
The collateral effect is obvious: we are losing the feeling about the J2EE concepts. Most of new Java developers share a weak idea about what is a Servlet, what is J2EE and other core concepts.

Of course experienced developers use these tools for enhancing their productivity and also every new technology has its own benefits to the Java community. But we think that is very important to keep an open mind, i.e., grant the people the choice of thinking differently while learning and discussing the new techniques. That´s the original idea of this project:

Learning the current techniques while discussing the Java basics through innovative thinking.
